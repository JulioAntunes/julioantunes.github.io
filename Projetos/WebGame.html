Web Game

Scene Investigator

Synopsis

The project revolves around the development of a browser-based game experience titled "Scene Investigators." Developed with HTML, CSS and Javascript, it is a demonstrative version of an upcoming game, featuring a narrative set during a celebratory dinner where an unexpected murder occurs.

Players explore various divisions of a house, collecting clues to solve the mystery using a computer interface. The primary goal is to offer users an immersive, non-linear narrative, challenging them to analyze clues, solve puzzles, and make decisions shaping the game's progression.

The project adopts an experimental design approach, breaking away from conventional interface design to deliver a unique and innovative experience. The non-linear narrative allows each player to have a personalized journey, unraveling the mystery in distinct ways. Careful consideration of interface design principles, multimedia elements, and interactive features contributes to an engaging and visually appealing experience.

The development involved various tasks, including narrative adaptation, 3D scenario creation, asset preparation, diagram creation, sound research, code production, and utilization of software such as Blender, Illustrator/Photoshop, and Visual Studio Code.

The project's technical foundation includes a carefully constructed diagram for spatial organization and orientation. 3D scenarios and assets were researched, adapted, and textured for a cohesive narrative. The HTML interface integrates interactive elements, animations, and a tutorial, guiding users through the game. Usability tests and iterations were conducted to ensure a quality user experience.

Reflection

The project faced challenges in narrative conception, visual component production, and HTML, CSS, and JavaScript programming. Extensive research and problem-solving were crucial, leading to the enhancement of technical skills. Despite the ambitious nature of the project, the team successfully delivered an innovative, immersive browser game experience, showcasing dedication, perseverance, and a commitment to continuous learning.
